# Amit Kamath Personal Website

This is a static personal website for Amit Kamath's job search profile.

## Free Publishing Option

The simplest no-domain, no-hosting-cost option is GitHub Pages:

1. Create a GitHub repository.
2. Add these files to the repository.
3. In GitHub, open `Settings > Pages`.
4. Set the source to the main branch and root folder.
5. Use the free URL GitHub provides, usually `https://username.github.io/repository-name/`.

## Files

- `index.html` - site content and structure
- `styles.css` - visual design and responsive layout
- `script.js` - small scroll reveal interaction
